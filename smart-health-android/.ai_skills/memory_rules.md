# QUY TẮC BỘ NHỚ (Memory Protocol)

Để duy trì tính nhất quán cho dự án Smart Healthcare System, bạn (Antigravity) phải tuân thủ:
1. Luôn tự động ghi nhớ cấu hình phần cứng: Cảm biến té ngã dùng dữ liệu 3 trục. Chân tín hiệu: DUST(G33/G14), TOUCH(G15), BUZZER(G13). KHÔNG ĐƯỢC làm sai.
2. Khi tôi và bạn thống nhất một kiến trúc code mới hoặc sửa xong một bug khó, hãy nhắc tôi lưu nội dung đó vào một file `project_log.md` để bạn đọc lại vào lần sau.
