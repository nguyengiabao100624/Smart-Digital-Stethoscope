# Contributing to Smart Digital Stethoscope

First off, thank you for considering contributing to the Smart Digital Stethoscope project! 

## How to Contribute

1. **Fork the repository** to your own GitHub account.
2. **Clone the project** to your local machine.
3. **Create a new branch** for your feature or bug fix.
4. **Make your changes** following the code style guidelines.
5. **Commit your changes** with clear and descriptive messages.
6. **Push your branch** to your fork.
7. **Submit a Pull Request** (PR) using our PR template.

## Code Style
- **Android:** Please follow standard Kotlin style guides. Run Android Studio lint before submitting.
- **Embedded:** Follow C++ best practices. Keep the Edge AI inference footprint as minimal as possible.

## Reporting Bugs
If you find a bug, please create an Issue using our provided `Bug Report` template. Provide as much detail as possible (logs, screenshots, steps to reproduce).
