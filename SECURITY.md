# Security Policy

## Supported Versions

Currently, the following versions of the project are actively supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a high priority for our medical application.

If you discover any security vulnerability in this project, **do not open a public issue**. Instead, please send an email to the repository owners:
* Danh Nguyen: danhnptho.word284@gmail.com
* Bao Nguyen: nguyengiabao100624@gmail.com

We will review the report and provide a status update within 48 hours. Please provide detailed steps to reproduce the vulnerability.
